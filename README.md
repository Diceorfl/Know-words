# Know_words   
## Приложение проверяет насколько хорошо вы владеете каким-либо иностранным языком./The app checks how well you know any foreign language.    

### Для чего это?  
    Чем лучше мы владеем каким-либо языком, тем меньше трудностей у нас вызывает прочитать текст из слов с перемешанными буквами.  
    По рзеузльаттам илссоевадний одонго анлигсйокго унвиертисета, не иеемт занчнеия, в каокм проякде рсапжоолены бкувы в солве. Галовне     чотбы преавя и пслонедяя бквуы блыи на мсете. осатьлыне бкувы мгоут селдовтаь в плоонм бсепордяке, все-рвано ткест чтаитсея без      побрелм. Пичрионй эгото ялвятеся то, что мы не чиаетм кдаужю бкуву по отдльенотси, а все солво цлиеком.  
### Как это работает?  
 #### Управление:      
      Load Dictionary - загрузка словаря в формате txt.  
      About Dictionary - интересная информация о словаре.  
      size - размер шрифта.  
      SEARCH - поиск слова из Input в словаре и его вывод в Output.   
      SORT - перемешивает буквы в введенном вами слове.  
      RANDOM - выводит в Input случайное перемешенное слово из словаря.      
![](https://github.com/Diceorfl/Know-words/blob/main/about_dictionary.PNG?raw=true)  

 #### Что делать?  
      1.Нажимаем Load Dictionary и выбираем нужный нам словарь.  
      2.Жмем RANDOM  
      3.Подбираем комфортный шрифт.  
      4.Пытаемся угадать слово.  
      5.Жмем SEARCH  
      6.Получаем вывод всех возможных слов.   

 #### Требования для сборки проекта:  
      Комплект: Qt5.15.1 MSVC2015.  


### How does it work?  
 #### What are these buttons for?  
      Load Dictionary - loading the dictionary in txt format.    
      About Dictionary - interesting information about the dictionary.     
      size - font size.    
      SEARCH - search for a word from Input in the dictionary and its output.   
      SORT - mixes the letters in the word you entered.  
      RANDOM - outputs a random mixed word from the dictionary into Input.  
![](https://github.com/Diceorfl/Know-words/blob/main/about_dictionary.PNG?raw=true)

 ####  How to use it?  
       1.Press the Load Dictionary and select the dictionary we need.  
       2.Click RANDOM.  
       3.We select a comfortable font.  
       4.Trying to guess the word.  
       5.Click SEARCH.  
       6 We get all possible words in Output.   

 #### Requirements for building the project:  
      Qt5.15.1 MSVC2015.
